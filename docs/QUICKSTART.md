# KJV Words — Quick Start

Get the app running in under 2 minutes.

## Prerequisites

- **Node.js** 18+ ([nodejs.org](https://nodejs.org))
- **npm** (comes with Node)

## 3 steps

### 1. Install

```bash
cd kjv-words
npm install
```

### 2. Run

```bash
npm run dev
```

### 3. Open

In your browser: **http://localhost:3000**

---

## What you’ll see

- **Home** — List of 8 topics (Creation & Redemption, Faith & Holy Spirit, Tradition & Law, etc.).
- **Topic page** — Click a topic to see KJV verses with Korean summaries. Use them for sharing, teaching, or personal study.

## Build for production

```bash
npm run build
npm start
```
